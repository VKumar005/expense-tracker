# Expense Manager App

A Full-Stack Web Application to track, visualize, and analyze daily financial expenses. Built with Python (Flask) and SQLite, featuring interactive data visualization and cloud deployment.

## Live Demo
[Click here to view the Live App](https://p2-expense-tracker.onrender.com)

## Tech Stack
* Backend: Python, Flask
* Database: SQLite
* Frontend: HTML5, Bootstrap 5
* Visualization: Chart.js (Interactive Doughnut Charts)
* Deployment: Render (Cloud Hosting)

## Key Features
* Dashboard Analytics: Real-time calculation of Daily, Monthly, and Yearly spending.
* Interactive Visualization: Dynamic Chart.js integration to visualize spending.
* History Mode: Filter transactions by specific dates.
* Responsive Design: Works on Mobile and Desktop.
